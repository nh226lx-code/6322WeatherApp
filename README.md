# 6322WeatherApp - 天气查询应用

## 项目简介

**6322WeatherApp** 是一款基于 Web 的天气查询应用，允许用户查询任意城市的实时天气数据。用户可以查看城市的温度、天气类型、风速、湿度和空气质量等信息。此外，用户还可以收藏感兴趣的城市并为其添加备注，方便快速查看天气。

## 功能特点

- **查询天气**：输入城市名，查询该城市的当前天气信息，包括温度、天气类型、风速、湿度和空气质量。
- **收藏城市**：用户可以将查询到的城市加入收藏，收藏的城市会显示在下方，并提供快速查询。
- **添加备注**：用户可以为收藏的城市添加备注，帮助区分不同城市（例如：家乡、公司等）。
- **自定义提示框**：所有提示信息通过自定义的消息框显示，而非浏览器默认的弹窗，提升用户体验。

## 技术栈

- **前端**：HTML、CSS、JavaScript
- **API**：使用 [WeatherAPI](https://www.weatherapi.com/) 获取天气数据

## 项目结构

- **`index.html`**：项目的主页，包含天气查询界面。
- **`style.css`**：应用的样式文件，负责页面的布局和外观设计。
- **`weatherApi.js`**：负责处理天气数据查询和渲染逻辑。
- **`collection.js`**：实现城市收藏和备注功能，允许用户保存和管理城市数据。
- **`app.js`**：核心的应用逻辑，处理天气查询、收藏、备注编辑等功能。
