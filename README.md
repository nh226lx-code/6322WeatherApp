# 6322WeatherApp

## 项目介绍
6322WeatherApp 是一款达到商业级UI标准的天气查询应用，基于原生JavaScript + AJAX实现异步天气数据获取，支持城市天气查询、收藏、备注编辑、收藏删除等完整的增删改查功能。应用采用现代化设计语言，适配全终端设备，交互流畅、视觉精致、运行稳定。

## 核心特性
### 🎨 商业级UI设计
- 参考「墨迹天气/苹果天气」设计规范，精细化视觉细节；
- 渐变色彩、分层阴影、流畅动效，符合现代UI审美；
- 响应式布局，完美适配手机/平板/电脑；
- 统一的设计语言，视觉层级清晰，交互反馈友好。

### 🚀 核心功能
| 功能 | 说明 |
|------|------|
| 天气查询 | 输入城市名，异步获取实时温度、天气状况、风向、湿度、空气质量等数据 |
| 城市收藏 | 收藏常用城市，本地持久化存储（刷新不丢失） |
| 备注编辑 | 为收藏城市添加个性化备注（如「老家」「公司」） |
| 收藏删除 | 管理收藏列表，删除不需要的城市 |
| 异常处理 | 完善的错误提示、加载状态、空状态兜底 |

### 🛠 技术栈
- **HTML5**：语义化标签 + Font Awesome图标库集成；
- **CSS3**：CSS变量、Grid/Flex布局、渐变、动画、响应式适配；
- **JavaScript**：
  - Fetch API实现AJAX异步请求；
  - 本地存储（localStorage）持久化收藏数据；
  - 模块化代码结构，完善的异常处理；
  - 流畅的交互动效和用户反馈；
- **第三方API**：国家气象局公开天气接口（稳定、无密钥、无跨域）。

## 使用方法
### 1. 本地运行
- 环境要求：Chrome/Firefox/Edge/Safari等主流浏览器；
- 运行方式：直接打开`index.html`文件即可启动应用；
- 无需搭建服务器，零配置开箱即用。

### 2. 功能操作
1. **天气查询**：在搜索框输入城市名（如北京、上海、广州），点击「查询天气」按钮或按回车键；
2. **收藏城市**：查询天气后，点击「收藏该城市」按钮添加到收藏列表；
3. **编辑备注**：在收藏列表中点击「编辑」按钮，输入个性化备注；
4. **删除收藏**：点击「删除」按钮，确认后移除该城市收藏。
